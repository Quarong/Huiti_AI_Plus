慧题AI+ (Huiti AI+)
下一代智能学术题库管理系统 —— AI 驱动知识闭环，让学习更高效、更专业。
![alt text](https://img.shields.io/badge/Version-0.9.0-blue?style=for-the-badge)
🌟 软件简介
慧题AI+ 是一款专为学生等终身学习人群打造的高级学术辅助工具。它深度集成顶级大语言模型（如 Google Gemini 3, 字节跳动豆包, DeepSeek 等），实现了从“原始资料”到“深度解析题库”的一键转换。
无论是期末复习、考研备考还是职业认证，慧题AI+ 都能通过 AI 算法为您定制专属的练习体验。
🚀 核心功能
1. AI 智能一键出题 (Magic Generator)
多格式支持：支持直接上传 PDF、Word (Docx) 或 TXT 资料。
语义理解：AI 自动阅读资料并精准提取核心知识点，自动生成单选题、判断题、填空题及简答题。
深度解析：每道题目均附带超过 50 字的逻辑分析，帮助您“知其然更知其所以然”。
2. 智能学情看板 (Intelligence Dashboard)
多维能力模型：基于正确率与知识覆盖率生成学科雷达图。
薄弱项预警：系统自动识别您的盲点科目并发出预警，建议针对性练习。
动力曲线：实时追踪您的学习强度与进步轨迹，量化学习成果。
3. 专业自组试卷 & PDF 导出 (Exam Builder)
所见即所得：自由挑选题库中的题目，自定义试卷标题与规格。
全真排版：支持导出符合 A4 标准的专业 PDF 试卷。
附带答案页：导出文件包含标准试卷格式与详尽的参考答案页，方便纸质化自测。
4. 沉浸式模拟考场 (Mock Exam)
全真限时模拟：倒计时系统营造真实的考场压力。
AI 深度阅卷：即使是主观简答题，AI 也能根据标准答案给出语义级评分与反馈。
5. 灵活的 AI 引擎配置 (Model Settings)
多模型切换：支持 Google Gemini、豆包 (Ark)、DeepSeek、ChatGPT 等多种主流 AI 供应商。
参数自定义：可调节推理温度 (Temperature) 与 Top P 值，平衡出题的严谨性与创造力。
🛡️ 隐私与安全
您的数据，您做主。
100% 本地存储：所有题库、练习历史和试卷均存储在您浏览器的 LocalStorage 中，不设云端服务器。
无账号体系：无需注册，开箱即用，从物理层面隔离数据泄露风险。
全量备份：提供一键导出 JSON 备份功能，方便在不同设备间迁移数据。
🛠️ 技术栈
前端框架：React 19 (Hooks, Context API)
界面样式：Tailwind CSS (响应式设计, Bento Grid 布局)
图标系统：Lucide React
核心 AI：Google GenAI SDK (Gemini 3 Pro/Flash)
桌面端引擎：Tauri (Rust )
文档处理：Mammoth.js (Docx) & PDF.js
📦 安装与运行
Web 版
访问部署地址。
进入 “模型配置”，填入您的 API Key。
开始导入资料或手动录题。
桌面专业版 (Tauri)
下载并运行 慧题AI+_2.5.0_x64.msi。
享受更稳定的性能与系统级资源访问。
